[color=#bf307a][size=4]üì¶ Installation[/size][/color]
It is installed as a traditional SMF Theme for SMF 2.1.x, just grab the latest release of the package and go to: [i]Admin > Configuration > Themes and Layout > Install a New Theme > From an Archive > Select File > Upload[/i].

You can find detailed instructions and/or more documentation about the process in [url=https://wiki.simplemachines.org/smf/Installation_-_How_do_I_install_a_theme]Installation - How do I install a theme[/url].
